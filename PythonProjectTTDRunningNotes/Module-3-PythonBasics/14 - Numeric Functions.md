
In this section, we shall the functions that we can implement in the numerical 
These will be single line of code.
Since these are single line, we can use the print statements directly.


1. max()
	1. It gives us/returns the highest possible number present in the list.
		1. print(max(1,2,3,4,5))
			1. 5
```
print(max(1,2,3,4,5))
5
```

2. min()
	1. It gives us/returns the lowest possible number present in the list
		1. print(min(10,19,8,7,65))
			1. 7
```
print(min(10,19,8,7,65))
7
```

3. Absolute Value - abs()
	1. To find the absolute value of the number
		1. print(abs(-2020))
			1. 2020
```
print(abs(-2020))
2020
```

4. Power Of - pow()
	1. To find the power of the value
		1. print(pow(2,3))
			1. 8
```
print(pow(2,3))
8
```
