
In this section we are going to discuss about the for loop.
- What is a loop?
	- I would like to print 'a' - 10 times
	- Without any knowledge of loops, we will print -> print('a') 10 times
	- Lets take an example of list
		- l1 = ["mike", 10.2, 1980]
		- for i in l1
			- print(i)
	- For the 1st iteration, i represents mike - 2nd iteration, i represents 10.2 - 3rd iteration, i represents 1980

```
l = ["Mike", 10.2, 1980]  
for i in l:  
    print(i)  
  
x = "apple"  
for i in x:  
    print(i)
    
Output Response -
Mike
10.2
1980

a
p
p
l
e
```
