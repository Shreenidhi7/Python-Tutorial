
What is a language?
1. We talk to each other in English. So English is a language.
2. Its a medium in which we convey message to other person.
3. To communicate with humans, we need a medium - That's called a Language.
4. Similarly when we want to communicate with a computer, we need a medium - That's called a Programming Language

Programming Language
1. Its a medium to communicate with a machine/computer

Every Language will have its own words, grammar [Words & Grammar are 2 most important things of any language]

Similarly every programming language has something called "Keywords" or "Reserved words". We use these words to write programs
Examples of Keywords
- if
- for
- while
- return
- etc...

Similarly every programming language has its "own set of rules" or "grammar" - That is called "Syntax" [Syntax is nothing but a set of rules]

What is Python?
1. Python is one of the programming language
2. Python is used to communicate with the computer when we want to give some certain set of instructions to the computer
3. Python is called a "High Level Programming Language"
4. Its very easy and simple to use and learn
    * Easy to Debug and Troubleshoot if there is any issues
    * Syntax is pretty simple compared to other languages
    * It has a lot of libraries
        * Libraries - Its nothing but a set of that has been written and bundled together and these can be directly used to write programs or can be used in programs
        * Example - There are libraries related to machine learning, data science and AI
        * Example - If we want to create a "Graph" using python
	        * If we want to write it from scratch - We might have to write thousands of lines of code
	        * But if we use a library which is present in Python - We can write in 3 or 4 lines max to create a "Graph" using that library
	    * Large repositories of libraries are available in Python related to Data Science, Machine Learning, Game Development and others
5. Python is also used for Scripting and Automation, Integration with Cloud - Its a general purpose programming language.
6. Python also has a large community
	1. It means that lot of people are using python [The question could be solved easily]
	2. There is a lot of documentation for Python and for the Libraries in Python
